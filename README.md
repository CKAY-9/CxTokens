# CxTokens

*Plugin Version: 1.0 (Achieving More)*</br>
*Minecraft Version: 1.19+*

CxTokens is a token (money) system for Spigot/Paper Minecraft servers. 

## Features
- Lottery event
- Auction House
- Bounties
- Item Store
- Commands
- Local data storing (no need for a database)
- Highly configurable
